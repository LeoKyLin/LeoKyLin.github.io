Leetcode-122.买卖股票的最佳时机III

与昨天的这道题相似，只是要求最多只能进行两次交易。

动态规划：

可知一共有四种状态：

赋初值，x1 = -price[0] ; x2 = -price[0] ; y1 = 0 ; y2 = 0 ;

对于x1,这一天的票价比之前的高不卖出，也可能找到票价更低的股票；故x1 = max(x1,-price[i]);

对于y1,这一天可能卖出的价格更低，即不卖出;或者这一天票价更高要卖出y1 = max(y1,x1++price[i]);

对于x2,这一天的票价比之前的高不卖出，也可能找到票价更低的股票；故x2 = max(x2,y1-price[i]);

对于x2,这一天可能卖出的价格更低，即不卖出;或者这一天票价更高要卖出y2 = max(y2,x2+price[i]);
